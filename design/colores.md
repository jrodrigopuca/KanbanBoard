## Paleta de Colores

### Fondos y Elevaciones (Superficies)

Base / Fondo Abismal: #050505 (El fondo de toda la app. Casi negro, absorbe la luz).

Elevación 1 (Columnas / Modales base): #0A0A0A a #0D1117 (Para distinguir áreas grandes sin usar bordes pesados).

Elevación 2 (Tarjetas inactivas): #111111 a #141414 (El nivel estándar donde interactúas).

Elevación 3 (Tarjetas activas / Hover / Popovers): #1A1A1A a #1D2128 (Lo que está "más cerca" del usuario).

### Textos (Jerarquía de Legibilidad)

Primario (Títulos activos, Foco): #FFFFFF (Blanco puro, usar con moderación para máximo impacto).

Secundario (Títulos de tarjetas, Textos principales): #EEEEEE o #DDDDDD (Blanco hueso, reduce la fatiga visual al leer).

Terciario (Descripciones, Metadatos): #AAAAAA (Gris claro).

Cuaternario (Hints, Atajos de teclado, Textos inactivos): #888888 a #555555 (Gris oscuro, no compite por atención).

### Colores Semánticos (Estados y Etiquetas)

Son colores desaturados para no parecer un semáforo brillante, manteniendo la elegancia:

Acento Principal (Foco, Botones primarios, Enlaces): Azul #58A6FF (Un azul eléctrico pero suave).

Éxito / Hecho (Done): Verde #3FB950.

Peligro / Destructivo / Prioridad Alta: Rojo #F85149.

Advertencia / En Progreso / Ideas: Ámbar/Amarillo #E2B93B.

Gradiente Premium (Bordes activos / Drag & Drop): De #58A6FF a #8957E5 (Azul a Violeta).

#### Bordes y Divisores (Glassmorphism)

No uses colores sólidos para los bordes. Usa blanco con canal alfa (transparencia) para que absorban el color de fondo:

Divisores sutiles: rgba(255, 255, 255, 0.05)

Bordes de tarjetas: rgba(255, 255, 255, 0.08) a rgba(255, 255, 255, 0.15)
